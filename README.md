# continuity-test

check if a function is continuous (at a point) or not

it depends on [sympy](https://github.com/sympy/sympy) for basic functionality and [plotext](https://github.com/piccolomo/plotext) for plotting graphs

take a look at the [web version](https://github.com/karim-xyz/continuity-test-web)

---

## getting started

```bash
git clone https://github.com/karim-xyz/continuity-test
cd continuity-test

pip install -r requirements.txt # a venv is recommended

python main.py
```
